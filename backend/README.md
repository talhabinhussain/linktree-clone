# Linktree Clone Backend

This is the backend API for the Linktree Clone project, built with FastAPI, SQLModel (SQLAlchemy), and PostgreSQL.

## Deployment on Hugging Face Spaces

This repository is configured to deploy as a Docker container on Hugging Face Spaces.

- **SDK**: Docker
- **Exposed Port**: 7860
